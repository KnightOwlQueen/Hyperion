Fetch Prophetic Vision

The /vision endpoint allows developers to query the Hyperion ML engine for predictive data constellations based on historical solar intensity.

HTTP Request

POST https://api.hyperion.vision/v2/vision

Request Body

Parameter Type Description
cycle_id string The unique solar cycle hash to analyze.
depth integer Prophetic lookback (1-24 hours).
Example Request (cURL) 
curl -X POST https://api.hyperion.vision/v2/vision \
  -H "Authorization: Bearer ORACLE_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cycle_id": "HELIOS_99",
    "depth": 12,
    "fidelity": "TITAN"
  }'
Prophetic Response
{
  "status": "DIVINE_SUCCESS",
  "prediction": {
    "probability": 0.984,
    "next_spike": "14:00:05 UTC",
    "intensity": "MAXIMUM"
  }
}
